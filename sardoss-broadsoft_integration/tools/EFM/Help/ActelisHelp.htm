    NAVIGATING THE MetaASSIST VIEW INTERFACE


    The primary view in MetaASSIST View (MAV) contains 3 major sections:

    • Section A: A list of connected network elements
    • Section B: A list of components for the network element selected in Section A
    • Section C: Details of the component selected in Section B. Use this section to configure each component



    To navigate the MAV interface, click the network element you want to access in Section A. Then find and click the component you want to configure or view in Section B. Finally, look in Section C for details and configuration options of the component you selected. The example below shows how to view HSL-7







    CREATING A VLAN IN THE ML2300


    The VLANs in the ML2300 are partially pre-built. All you need to do is label the VLAN and tag the proper HSL.

    TROUBLESHOOTING










































    •

  • ML42 Help

    Two VLANS must be defined in the CPE gear: one for the upstream connection to the Cbeyond network and one for the customer's internal network. For ML42 EFM box, assign a management IP to the box through MetaASSIST View, then log into the device web interface using the management IP you assigned in a web browser. Remember to free up the management IP when done configuring the ML42. For the ML624 & ML628 EFM boxes, you can configure them directly from MetaASSIST View.
